On Vita nodes (c5.xlarge, four ENA):

 - Make sure src/dst check is disabled for *private* interface (eth1)!
 - Security groups must allow UDP/303, ESP, all traffic on private interface
 - ethtool -L eth1 combined 2 # private
 - ethtool -L eth2 combined 1 # public 1
 - ethtool -L eth3 combined 3 # public 2
